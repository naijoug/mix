# LLMS

## Usage

```sh
# activate virtual env
$ python -m venv .venv
$ source .venv/bin/activate

# install 
pip install tensorflow
pip install pytorch
pip install transformers
# error : [No module named 'keras.engine'](https://github.com/huggingface/transformers/issues/23352)

pip install fastapi
pip install uvicorn
```
