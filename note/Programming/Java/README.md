## JAX-WS vs JAX-RS

* JAX-WS (Java API for XML Web Services) : XML(SOAP/WSDL)
    > 动词为中心，指定每次执行的函数
- JAX-RS (Java API for RESTful Web Services) : JSON
    > 名称为中心，指定每次执行的资源
    * [Jersey](https://github.com/eclipse-ee4j/jersey) : JAX-RS 的一个实现
