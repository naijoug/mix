## ready vs onload

```javascript
// $(document).ready() : 页面的 DOM 结构加载完成 (不包含其它媒体资源)
$(document).ready(function() {
    /* code */
})
$(function() { // JQuery ready
    /* code */
})
// window.onload : 页面的所有资源加载完成 (包括 DOM 和其它媒体资源)
window.onload=function() {
    /* code */
 }
```
