## Concept

- `@discardableResult`
    > 可以去掉有返回值的函数，但是没有使用返回值的警告

- `@UIApplicationMain`
    > 替代 `OC` 项目中的 `main.m` 作为 `iOS` 程序的入口

- `Any` vs `AnyObject`
    * `Any` : Any 可以代表任何类型( struct、class、func 等等 )
    * `AnyObject` : 只能代表 class 生成的实例

- `as` & `as!` & `as?`
    * as : 转换
    * as! : 强制转换，失败会出现运行时错误
    * as? : 不成功返回 `nil`，成功返回可选类型
