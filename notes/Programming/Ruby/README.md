## Usage

```bash
$ sudo gem update --system  # 更新ruby
$ gem sources -l            # 显示当前sources
$ gem sources --add https://gems.ruby-china.com/   # 添加镜像source
$ gem sources --remove https://rubygems.org/    # 删除镜像source
```
